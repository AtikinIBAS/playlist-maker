# Playlist Maker

Учебный Android-проект на Jetpack Compose, который постепенно собирался по спринтам.

## Что реализовано

- главный экран приложения
- экран поиска
- экран настроек
- экран списка треков
- базовая навигация между экранами
- поиск треков по названию и исполнителю
- архитектурное разделение на слои `Data / Domain / UI`
- `MVVM` для экрана поиска

## Стек

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- AndroidX Lifecycle ViewModel

## Конфигурация проекта

- Android Studio: проверялось в Android Studio с JetBrains Runtime 21
- JDK: 21 (`jbr-21`)
- `compileSdk = 36`
- `targetSdk = 36`
- `minSdk = 29`

## Как собрать и запустить

1. Откройте проект в Android Studio.
2. Убедитесь, что для Gradle выбран `Gradle JDK = jbr-21` или другой JDK 17+.
3. Дождитесь завершения `Gradle Sync`.
4. Выберите эмулятор или подключённое устройство.
5. Нажмите `Run`.

## Проверка сборки

Для локальной сборки можно использовать:

```powershell
.\gradlew.bat assembleDebug
```

## Основные ветки

- `sprint-3` — главный экран и базовые переходы между экранами
- `sprint-4` — навигация на Compose, экран поиска и экран настроек
- `sprint-5` — архитектурная подготовка поиска и список треков
- `sprint-6` — полноценный поиск на `MVVM`
