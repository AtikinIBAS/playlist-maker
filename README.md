# Playlist Maker

Учебный Android-проект на Jetpack Compose. Приложение позволяет искать треки, добавлять их в избранное, создавать плейлисты, прикреплять к ним обложки и удалять треки из плейлиста.

## Что реализовано

- главный экран приложения;
- экран поиска с историей, состояниями загрузки, ошибки и пустого результата;
- экран настроек с системными интентами;
- экран избранных треков;
- экран списка плейлистов;
- экран создания плейлиста с выбором обложки из галереи;
- экран деталей трека;
- экран деталей плейлиста;
- добавление треков в плейлист и удаление трека из плейлиста по долгому нажатию;
- архитектурное разделение на слои `Data / Domain / UI`;
- `MVVM` для экранов с бизнес-логикой.

## Технологии

- Kotlin `2.0.21`
- Android Gradle Plugin `8.13.2`
- Jetpack Compose
- Material 3
- Navigation Compose
- Lifecycle ViewModel
- Retrofit
- Room
- DataStore

## Конфигурация проекта

- Android Studio: рекомендуется актуальная версия с JetBrains Runtime 21
- JDK: 21 (`jbr-21`)
- `compileSdk = 36`
- `targetSdk = 36`
- `minSdk = 29`
- `jvmTarget = 11`

## Сборка и запуск

1. Откройте проект в Android Studio.
2. Убедитесь, что для Gradle выбран `Gradle JDK = jbr-21` или другой совместимый JDK 21.
3. Дождитесь завершения `Gradle Sync`.
4. Выберите эмулятор или подключённое устройство.
5. Запустите приложение кнопкой `Run`.

Для проверки сборки из терминала:

```powershell
.\gradlew.bat assembleDebug
```
