# Sprint 12 Checklist

Финальная самопроверка перед отправкой проекта на ревью.

## Общая проверка

- [x] Проект компилируется и запускается без ошибок.
- [x] При запуске открывается главный экран.
- [x] Навигация между экранами работает стабильно, приложение не падает.
- [x] Навигация задаётся в классе `PlaylistHost`.

## Главный экран

- [x] На главном экране есть кнопки для перехода к поиску, плейлистам и настройкам.
- [x] При нажатии на кнопки выполняются переходы на соответствующие экраны.
- [x] В манифесте корректно прописаны все Activity.

## Экран поиска

- [x] В строке поиска отображается placeholder «Поиск».
- [x] Есть иконка «Лупа» слева и иконка «Очистить» справа.
- [x] При вводе текста можно очистить строку нажатием на «х».
- [x] При нажатии на «Лупу» запускается поиск.
- [x] Пока поиск идёт, показывается индикатор загрузки.
- [x] Если поиск завершён успешно, отображается список треков.
- [x] Если произошла ошибка, показывается сообщение об ошибке.
- [x] Если по запросу ничего не найдено, показывается пустое состояние.
- [x] Отображается длительность трека в формате `mm:ss`.
- [x] Список треков верстается с помощью `LazyColumn`, элементы списка кликабельны.

## Экран настроек

- [x] Есть три кнопки: «Поделиться приложением», «Написать разработчикам», «Пользовательское соглашение».
- [x] Нажатие «Поделиться» открывает системное окно для выбора приложения.
- [x] Нажатие «Написать разработчикам» открывает почтовый клиент с заполненными адресом, темой и телом письма.
- [x] Нажатие «Пользовательское соглашение» открывает браузер со ссылкой.
- [x] Все пользовательские строки вынесены в ресурсы.

## Архитектура поиска

- [x] Архитектура разделена на `Data / Domain / UI`.
- [x] Реализован `Storage` как эмулятор данных.
- [x] Реализован репозиторий `TracksRepositoryImpl`.
- [x] В `Domain` есть интерфейс `TracksRepository`, в `Data` — `NetworkClient`.
- [x] `SearchViewModel` управляет состояниями `Initial`, `Searching`, `Success`, `Fail`.
- [x] `SearchScreen` корректно отображает состояния поиска.

- [x] Нажатие на кнопку вызывает действие.
